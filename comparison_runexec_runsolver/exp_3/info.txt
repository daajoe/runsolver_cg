The experiment is run on for testing memory (with swap) on runexec, with the memory limit being 8GB (8589934592). It is repeated 50 times. 
The command used for runexec is: runexec --memlimit 8589934592 python3 test_memory.py
The command used for runsolver is: ./runsolver -M 8192 python3 test_memory.py
